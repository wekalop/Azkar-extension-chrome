# 🕌 Azkar Chrome Extension

A lightweight Chrome extension that provides daily Islamic azkar reminders and accurate prayer times based on your location.

---

## ✨ Features

- 📿 Random Azkar Popups  
  Displays periodic Islamic reminders (Azkar) selected from a curated dataset.

- 🕌 Prayer Time Widget  
  Floating widget showing:
  - Next prayer
  - Live countdown

- 📍 Location-Based Timing  
  Uses GPS or manual city input to fetch accurate prayer times.

- ⏰ Smart Prayer Alerts  
  Visual + sound reminder 15 minutes before prayer.

- 🎛️ Customizable Settings  
  - Adjust azkar interval  
  - Enable/disable sound  
  - Move widgets freely  

---

## 🛠️ Tech Stack

- JavaScript (Vanilla)
- Chrome Extension (Manifest V3)
- Aladhan Prayer Times API
- HTML / CSS
