# AzkarV1 Store Listing Template

## Name
AzkarV1 - Prayer Time & Azkar Reminder

## Short value proposition
A lightweight Islamic reminder extension with random Azkar popups and a compact next-prayer countdown widget.

## Detailed Description
AzkarV1 helps users stay connected with daily remembrance and prayer awareness while browsing.

Core features:
- Random Azkar popup reminders at configurable intervals.
- Popup content selected from curated Azkar IDs (52-128).
- Compact floating widget showing next prayer and live countdown.
- 15-minute pre-prayer alert with visual warning and short sound.
- GPS-first location detection with manual city/country fallback.
- Daily prayer times fetched from Aladhan API (method=3) and cached locally.
- Draggable widget and popup with responsive Arabic-friendly UI.

Privacy-first behavior:
- Uses local storage for settings and cached prayer times.
- No login required.
- No ad tracking.
